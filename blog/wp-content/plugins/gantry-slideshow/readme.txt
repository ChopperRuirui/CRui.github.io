Slideshows made easy!

== Description ==

Integrated with Gantry's override system, provides a slick interface to build a slideshow. Requires <a href="http://gantry-framework.org">Gantry Template Framework</a>.

== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Appearance -> Slideshow and add some slides
4. Goto Appearance -> Widgets and add a Slideshow widget where you want it to appear
5. Enjoy!
